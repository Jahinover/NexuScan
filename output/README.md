# NexuScan: PDF Automator 📄🚀

**NexuScan** es una herramienta de ingeniería diseñada para optimizar flujos de trabajo logísticos. Automatiza la extracción de códigos de movimiento desde documentos PDF escaneados, genera enlaces de acceso rápido y divide los archivos en documentos individuales organizados.

## ✨ Funcionalidades
- **OCR Automatizado:** Extrae códigos que inician con "032" usando Tesseract OCR.
- **Flujo Continuo:** Procesamiento automático al cargar el archivo (sin botones innecesarios).
- **Generador de URLs:** Crea enlaces directos a los movimientos detectados.
- **Segmentación Inteligente:** Divide el PDF original en Manifiestos y Remesas con nombres normalizados.
- **Exportación en ZIP:** Descarga todos los archivos procesados en un solo paquete comprimido.
